Grok3使用入口和订阅教程(白嫖150刀API额度)

前言介绍

重点：Grok 3仅对X上的Premium+用户开放。你需要订阅X的Premium+服务才能访问Grok 3的功能。

也就是说，你需要注册一个X账号（原推特）。并且升级到对应的版本才能有权限访问 Grok 3。

目前可以在 IOS 的 APP Store 上进行安装 。

教程开始之前，我们先来了解一下 Grok 的 网页版官方使用地址：

web网页版：https://grok.com/

非常简洁，是不是非常熟悉！有点DeepSeek的熟悉味道了~~~

Grok不仅在官网上使用，你也可以在 X（原推特）上进行使用。打开 X 之后，（通常在界面底部或侧边栏）上就能看到 Grok。

X （原推特）上的网页版：https://x.com/i/grok

如何订阅X上Premium+服务

国内订阅X的Premium+教程

申请一个X账号，你可以通过Google邮箱进行注册 X 也行。然后再用 X 进行登录。

X（原推特）官网：https://x.com/

进入 X 官网之后，点击侧边栏上的 “Premium”，就可以进入订阅页面。

按年订阅有优惠，不过正常我们都是按月进行订阅。费用：326元人民币。

Grok 3 公开的基准上来看，超越了 DeepSeek-R1、Gemini、o1、o3-mini（high）。 从价格上和ChatGPT对比的话，肯定不贵啊！

而且，Grok 3 是 首个超过 1400 分 的模型啊！果然，马斯克没有吹牛，Grok3 是世界上最聪明的 AI。

白嫖教程

在点击下方 “Subscribe & Pay” 之前，我们先申请一张虚拟信用卡，再进行订阅和支付。

因为外国人的消费习惯，都是先预支的。所以大部分付款都是使用信用卡。而国内用户，大部分也没有master、visa国际信用卡。

但是我们可以借助 “master 虚拟信用卡” 解决。 下面打卡比较慢时，挂上魔法可以加速。

虚拟卡申请：https://wildscard.com

虚拟信用卡平台，支持订阅支付的 X 和 其他平台服务。

如果不着急的伙伴，可以让Grok3的子弹再飞一会，看看大家的实测再决定订阅Grok平台上的SuperGrok。

你也可以先白嫖一下 Grok API的福利，每月赠送 150 刀体验金，前提是要先给自己的API账号充值5美刀先。

白嫖之前，肯定得先知道Grok API平台，并且注册一个账号，建议使用谷歌邮箱”登录，下面就是Grok API 开发者平台：

https://console.x.ai/

而目前 Grok API 提供的模型还没有Grok 3，只有 Grok 2。 Grok 3 发布会直播时，声称需要等待几周后才开始推送 Grok 3 的 API。

那么如何才可以薅到Grok API 的 150美元体验金呢？有个前置条件：

  • 绑定卡，并充值最低 5 美元
  • 加入数据共享计划

国内如何绑卡并充值 Grok API 5 美元呢？

如果你有国内的visa、master国际信用卡，也可以尝试。如果不行，或者没有国际信用卡的话，就使用Master虚拟信用卡。

Master虚拟信用卡：https://wildscard.com

申请之后，会有一个美区的Master国际信用卡信息，可以把这些信息进行绑定 X AI 平台，也就是 Grok API 平台。

先给虚拟卡充值5美刀，可以使用支付宝进行充值，不使用的话，可以直接提现。

绑卡时，注意使用的魔法建议纯净度良好的，自测方式：ping0.cc。 绑定时，被拒的原因，大部分是网络环境问题。

打开 Grok API 平台：console.x.ai 按下面，找到 “Payment settings” 进行 填写账单地址，然后再添加卡。

这两个信息都直接用虚拟卡上的信息即可。虚拟卡上没有的信息，可以不用填，比如 最下方的电话号码。

填完账单地址之后，就可以进行绑卡了，绑定之后，充值 5 美元即可。

这里，我使用的魔法节点是美区｜原生｜IP，纯净度还可以。就直接一次性过了，不通过的情况下，可以重试哈。

绑定成功之后，有一条预扣款 0 元的信息：

确保自己的虚拟卡账户有 5 美元，最后一步：现在你可以进行一个充值5美元的操作了。

找到 “Credits”，然后往下滑动， 找到 “Purchase credits” 进行支付即可。

成功之后，开启数据共享协议，就可以有下面这个150美元的免费额度啦～

如果，你充值了 5 美元之后，还没有到账 150 美元的话，可以来这里查 Free Credits via Data Sharing | xAI Docs，因为需要签署一个数据共享协议:

你充值5美元之后，在下方测试是否有资格的，可以点击 “check for eligibility“ 就行，之后再到 ”credit“ 里面选择共享数据就可以获得150美元的额度：

小结

要使用Grok 3，你需要：

订阅X的Premium+服务：

  • Grok 3仅对X上的Premium+订阅者开放。你需要在X上订阅Premium+才能访问Grok 3的功能。

通过X使用：

  • 订阅后，打开X应用或网站。寻找Grok图标（通常在界面底部或侧边栏），点击它进入Grok 3的聊天界面。直接在这里提问或使用其功能。

SuperGrok订阅（可选）：

  • 如果你希望获取更多高级功能，例如通过Grok网站或应用进行更深入的交互，可能需要额外订阅SuperGrok。SuperGrok提供对Grok 3的额外功能和更高级的使用体验。

功能使用：

  • 正常模式：用于日常查询。
  • 研究模式（DeepSearch）：对于需要深入研究的问题，使用此模式允许Grok在网络和X上搜索信息。
  • 大脑模式（Big Brain Mode）：为复杂问题提供详细的推理过程。

订阅支付解决方法

  • 虚拟信用卡申请：解决支付问题，官网：https://wildscard.com
